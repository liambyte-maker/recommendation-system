# 优化器详解释


# 1. 背景

优化器的作用：

深度学习中，优化器用于调整模型的参数（权重，偏置，embedding),来使得损失函数最小

鞍点在一个方向上是最低点， 但是在另一个方向上，是最高点

因为深度学习神经网络，参数维度非常高，在高空间，鞍点数量远远多于真正的局部最小值

梯度下降会认为已经收敛，但其实不是最低点

## 区分鞍点和最小值

看二阶导数，

如果所有方向曲率都>0, 最小值

如果有的方向>0, 有的<0, 鞍点


## 2. 常考优化器机器考点

梯度下降类

最常见的一类优化器，通过计算loss相对于模型参数的梯度，沿着梯度下降的方向更新参数，以最小化损失函数

SGD(stochastic Gradient Descent 随机梯度下降）
